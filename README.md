# Build
Get the source on your local
```sh
git clone https://github.com/pearl-lang/pearl.git && cd pearl
```

now build the source
```sh
mkdir build && cd build
cmake ..
make all
```

or via build script, also you can build a release with "-r" argument
```sh
sh build.sh -b
```
